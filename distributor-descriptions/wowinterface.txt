[SIZE=6][COLOR="#58be81"]About Simple Quest Plates[/COLOR][/SIZE]

[COLOR="#58be81"][B]SQP | Simple Quest Plates![/B][/COLOR] is a professional World of Warcraft addon that displays quest progress icons directly on enemy nameplates. Never wonder if that mob is for your quest again! This lightweight addon seamlessly integrates with the default UI to show quest objectives right where you need them.

As part of the [COLOR="#8B1538"][B]RGX Mods[/B][/COLOR] collection, SQP is actively maintained and supported by our vibrant [URL="https://discord.gg/N7kdKAHVVF"][COLOR="#7289DA"][B]Discord community[/B][/COLOR][/URL] of over 5,000 WoW players. Join us for instant support, feature requests, and to connect with fellow adventurers!

[SIZE=6][COLOR="#58be81"]Key Features[/COLOR][/SIZE]

[SIZE=5][COLOR="#4ecdc4"]Quest Detection System[/COLOR][/SIZE]
[LIST]
[*][B]Automatic Quest Tracking[/B] - Instantly identifies quest mobs and objectives
[*][B]Smart Priority System[/B] - Shows item requirements before kill counts when both are needed
[*][B]World Quest Support[/B] - Full compatibility with world quests and bonus objectives
[*][B]Multi-Quest Tracking[/B] - Handles multiple quests requiring the same target
[*][B]Progress Numbers[/B] - Clear display of remaining kills or items needed
[*][B]Real-Time Updates[/B] - Icons update immediately as you complete objectives
[/LIST]

[SIZE=5][COLOR="#4ecdc4"]Complete Customization System[/COLOR][/SIZE]
[LIST]
[*][B]Icon Positioning[/B] - Place icons on left or right of nameplates with pixel-perfect offset control
[*][B]Icon Scaling[/B] - Adjust size from 50% to 200% of default
[*][B]Font Customization[/B] - Choose font size (8-24), outline style (None/Normal/Thick)
[*][B]Color Options[/B] - Separate colors for kill quests, item quests, and progress quests
[*][B]Icon Tinting[/B] - Optional color overlay to match your UI theme
[*][B]Outline Colors[/B] - Customize text outline color for better visibility
[/LIST]

[SIZE=5][COLOR="#4ecdc4"]Professional Options Panel[/COLOR][/SIZE]
[LIST]
[*][B]Tab-Based Interface[/B] - Organized settings in General, Font, Icon, and About tabs
[*][B]Live Preview System[/B] - See changes instantly with the built-in preview
[*][B]Reset Options[/B] - Individual and global reset buttons for easy management
[*][B]Settings Profiles[/B] - Account-wide settings that sync across all characters
[*][B]Smooth Scrolling[/B] - Navigate long option lists with mouse wheel support
[*][B]Professional Design[/B] - Clean, modern interface matching Blizzard standards
[/LIST]

[SIZE=5][COLOR="#4ecdc4"]Performance and Compatibility[/COLOR][/SIZE]
[LIST]
[*][B]Ultra Lightweight[/B] - ~200KB memory footprint
[*][B]Optimized CPU Usage[/B] - < 0.1% impact during normal gameplay
[*][B]Error Resilient[/B] - Protected calls prevent addon conflicts

[SIZE=6][COLOR="#58be81"]Slash Commands[/COLOR][/SIZE]

[CODE]
/sqp              - Open the options panel
/sqp help         - Display all available commands
/sqp on           - Enable the addon
/sqp off          - Disable the addon
/sqp test         - Run quest detection test
/sqp status       - Show current settings
/sqp scale <num>  - Quick adjust icon scale (0.5-2.0)
/sqp offset <x> <y> - Quick adjust icon position
/sqp reset        - Reset all settings to defaults
[/CODE]

[SIZE=6][COLOR="#58be81"]Installation Guide[/COLOR][/SIZE]

[SIZE=4][B]Step-by-Step Installation:[/B][/SIZE]
[LIST=1]
[*]Download Simple Quest Plates from this page
[*]Close World of Warcraft completely
[*]Extract the downloaded ZIP file
[*]Copy the [COLOR="#4ecdc4"]SimpleQuestPlates[/COLOR] folder to:
   [COLOR="#4ecdc4"]World of Warcraft\_retail_\Interface\AddOns\[/COLOR]
[*]Ensure the folder is named exactly: [COLOR="#4ecdc4"]SimpleQuestPlates[/COLOR]
[*]Launch World of Warcraft
[*]At character select, click "AddOns" button
[*]Verify "Simple Quest Plates" appears and is checked
[*]Enter the game
[*]Type [COLOR="#4ecdc4"]/sqp[/COLOR] to open configuration panel
[/LIST]

[SIZE=4][B]Quick Start:[/B][/SIZE]
The addon works immediately upon installation! Just enable enemy nameplates (V key) and quest icons will appear automatically.

[SIZE=6][COLOR="#58be81"]Troubleshooting Guide[/COLOR][/SIZE]

[SIZE=4][COLOR="#e67e23"][B]Icons Not Appearing?[/B][/COLOR][/SIZE]
[LIST]
[*]Ensure enemy nameplates are enabled (V key by default)
[*]Check that the addon is enabled with [COLOR="#4ecdc4"]/sqp on[/COLOR]
[*]Verify you have active quests with trackable objectives
[*]Try [COLOR="#4ecdc4"]/sqp test[/COLOR] to verify detection is working
[*]Check if "Hide in Combat" or "Hide in Instance" options are enabled
[*]Ensure quest tracking is enabled in your quest log
[/LIST]

[SIZE=4][COLOR="#e67e23"][B]Performance Issues?[/B][/COLOR][/SIZE]
[LIST]
[*]Disable "Hide in Combat" if experiencing combat lag
[*]Reduce icon scale for better performance
[*]Check for conflicts with other nameplate addons
[*]Try disabling other addons temporarily to isolate issues
[*]Use [COLOR="#4ecdc4"]/sqp debug[/COLOR] to check for errors
[/LIST]

[SIZE=4][COLOR="#e67e23"][B]Settings Not Saving?[/B][/COLOR][/SIZE]
[LIST]
[*]Ensure the addon has write permissions
[*]Check that SavedVariables folder exists
[*]Try [COLOR="#4ecdc4"]/reload[/COLOR] after making changes
[*]Use [COLOR="#4ecdc4"]/sqp reset[/COLOR] if settings appear corrupted
[*]Verify the addon folder name is exactly "SimpleQuestPlates"
[/LIST]
