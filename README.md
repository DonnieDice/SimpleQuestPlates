## <span style="color:#ff6b6b">🎯 What is SQP?</span>

**<span style="color:#58be81">S</span><span style="color:#58be81">Q</span><span style="color:#58be81">P</span> <span style="color:#fff">|</span> <span style="color:#58be81">S</span><span style="color:#fff">imple </span><span style="color:#58be81">Q</span><span style="color:#fff">uest </span><span style="color:#58be81">P</span><span style="color:#fff">lates!</span>** <span style="color:#e67e23">enhances your</span> <span style="color:#06c">World of Warcraft</span> <span style="color:#e67e23">experience by displaying quest progress icons directly on enemy nameplates. Know at a glance which enemies you need to defeat for your quests, how many items they drop, and your overall progress — all without cluttering your interface.</span>

- **<span style="color:#2dc26b">🎯 Quest Awareness:</span>** <span style="color:#e67e23">See quest-relevant enemies at a glance without checking your quest log</span>
- **<span style="color:#58be81">🎨 Fully Customizable:</span>** <span style="color:#e67e23">Adjust colors, fonts, sizes, and positions to your liking</span>

<a id="command-reference"></a>
## <span style="color:#ff6b6b">📋 Command Reference</span>

<span style="color:#e67e23">Use</span> <span style="color:#2dc26b">`/sqp`</span> <span style="color:#e67e23">followed by</span><span style="color:#3598db">:</span>

<div align="center">

| Command | Description |
|---------|-------------|
| `/sqp` or `/sqp options` | Open the options panel |
| `/sqp help` | Display all available commands |
| `/sqp on` | Enable the addon |
| `/sqp off` | Disable the addon |
| `/sqp status` | Show current settings |
| `/sqp version` | Show addon version |
| `/sqp test` | Test quest detection on current nameplates |
| `/sqp scale <0.5-2.0>` | Set icon scale |
| `/sqp offset <x> <y>` | Set icon position offset |
| `/sqp anchor <LEFT\|RIGHT>` | Set which side of nameplate to anchor to |
| `/sqp reset` | Reset all settings to defaults |
| `/sqp debug` | Toggle debug mode |

</div>

## <span style="color:#ff6b6b">🛠️ Configuration Tips</span>

<table width="100%">
<tr>
<td width="50%" valign="top">

### <span style="color:#4ecdc4">Recommended Settings:</span>
- **<span style="color:#2dc26b">Combat Players</span><span style="color:#3598db">:</span>** <span style="color:#e67e23">Enable "Hide in Combat" for cleaner fights</span>
- **<span style="color:#b96ad9">Dungeon/Raid</span><span style="color:#3598db">:</span>** <span style="color:#e67e23">Enable "Hide in Instances" to reduce clutter</span>
- **<span style="color:#58be81">Questing Focus</span><span style="color:#3598db">:</span>** <span style="color:#e67e23">Larger scale (1.5x) and bright colors for maximum visibility</span>
- **<span style="color:#ff6b6b">Minimalist</span><span style="color:#3598db">:</span>** <span style="color:#e67e23">Smaller scale (0.8x) with subtle positioning</span>

### <span style="color:#4ecdc4">Customization Options:</span>
- **<span style="color:#2dc26b">Icon Scale</span><span style="color:#3598db">:</span>** <span style="color:#e67e23">0.5x to 2.0x size adjustment</span>
- **<span style="color:#b96ad9">X/Y Offset</span><span style="color:#3598db">:</span>** <span style="color:#e67e23">-50 to +50 pixel positioning</span>
- **<span style="color:#58be81">Anchor Side</span><span style="color:#3598db">:</span>** <span style="color:#e67e23">Left or right side attachment</span>
- **<span style="color:#ff6b6b">Font Size</span><span style="color:#3598db">:</span>** <span style="color:#e67e23">8 to 20 point with outline and shadow options</span>
- **<span style="color:#4ecdc4">Color Options</span><span style="color:#3598db">:</span>** <span style="color:#e67e23">Different colors for kill, item, and progress quests</span>

</td>
<td width="50%" valign="top">

### <span style="color:#4ecdc4">How It Works:</span>
- <span style="color:#e67e23">SQP uses Blizzard's quest API to detect which enemies are related to your active quests</span>
- <span style="color:#e67e23">Automatically scans all nameplates in view</span>
- <span style="color:#e67e23">Updates in real-time as you accept or complete quests</span>
- <span style="color:#e67e23">Prioritizes item drops over kill counts for dual-objective quests</span>

### <span style="color:#4ecdc4">Performance:</span>
- <span style="color:#e67e23">Efficient caching system for quest data</span>
- <span style="color:#e67e23">Minimal CPU usage with smart update cycles</span>
- <span style="color:#e67e23">No impact on frame rates</span>

</td>
</tr>

## <span style="color:#ff6b6b">🔧 Troubleshooting</span>

**<span style="color:#ff6b6b">Quest icons not showing?</span>**
- <span style="color:#e67e23">Use</span> `/sqp test` <span style="color:#e67e23">to verify quest detection on current nameplates</span>
- <span style="color:#e67e23">Check if "Hide in Combat" or "Hide in Instances" is enabled via</span> `/sqp status`
- <span style="color:#e67e23">Confirm the addon is enabled with</span> `/sqp on`

**<span style="color:#ff6b6b">Settings not applying?</span>**
- <span style="color:#e67e23">Enable Debug Mode with</span> `/sqp debug` <span style="color:#e67e23">for detailed logging</span>
- <span style="color:#e67e23">Reset all settings to defaults with</span> `/sqp reset`
